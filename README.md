# WebsiteScraper
A website scraper (Node.js)

### Installing
Install dependencies :
```
$ sudo npm install website-scraper
```

### How to use?
```
$ node scrap.js [url]
```
You need to include protocole in url (http:// or https://).
The folder will be created where the script was launched.
